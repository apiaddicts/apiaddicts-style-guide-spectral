Noncompliant Example (OpenAPI 2):
---------------------------------

```yaml
swagger: "2.0"
paths:
  /resources:
    put:
      responses:
        200:
          description: Ok
```

Compliant Example (OpenAPI 2):
-------------------------------

```yaml
swagger: "2.0"
paths:
  /resources/{id}:
    put:
      responses:
        200:
          description: Ok
```

Noncompliant Example (OpenAPI 3):
---------------------------------

```yaml
openapi: "3.0.1"
paths:
  /users/get:
    post:
      responses:
        200:
          description: OK
```

Compliant Example (OpenAPI 3):
-------------------------------

```yaml
openapi: "3.0.1"
paths:
  /users:
    get:
      responses:
        200:
          description: OK
```
