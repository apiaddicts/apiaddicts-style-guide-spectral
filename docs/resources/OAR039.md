Noncompliant Example (OpenAPI 2):
---------------------------------

```yaml
swagger: '2.0'
info:
  version: 1.0.0
  title: Swagger Petstore
paths:
  /users:
    get:
      summary: Get users
      responses:
        200:
          description: Successful response
        400:
          description: Bad request
```

Compliant Example (OpenAPI 2):
-------------------------------

```yaml
swagger: '2.0'
info:
  version: 1.0.0
  title: Swagger Petstore
paths:
  /users:
    get:
      summary: Get users
      responses:
        200:
          description: Successful response
        400:
          description: Bad request
        500:
          description: Internal server error
        503:
          description: Service unavailable
```
